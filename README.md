# Mebelka.ua

INSTALLATION
------------

  1) Please, open your directory "xampp"
  2) Delete your directory "htdocs"
  3) Clone this project into directory "xampp"


Start working
------------

  1) Please, open  your directory "xampp"
  2) Start your local server  by clicking "xampp_start.exe"
  3) Open your browser
  4) Write in URL "localhost"
